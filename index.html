ZeroTalk-Go

Introduction

ZeroTalk is a fully automated go training system based on the paper alphago-zero.  It consists of three parts, MCTS, unified policy and value neural network, self-play. Network evaluation and self-play are executed in separate threads interactively in same process. The network training part is put in another process. At the end of self-play game stage, the train data is generated. Network training process receives the data location and starts training, every 2000 optimization steps, it generates checkpoint and send its location to self-paly process.


MCTS

MCTS is performed in during each step of self play game. First, from root state of current game step, it select recursively a move that maximize the UCT value, until a leaf node is met. Second, for the leaf node, it generates all legal moves and creates children for these moves. Third, the leaf node is evaluated using trained neural net model, generating moves distribution and value measuring the win probability. At the end of MCTS, the value is propogated up to root state.


Self-Play

At each slef play step, the program select the move(node) proportional to number of times visited generated from MCTS.


Deep Neural Network

During MCTS stage, each leaf node is evaluated by neural network, generating moving probability and state value. The training data is generated from self play games.
